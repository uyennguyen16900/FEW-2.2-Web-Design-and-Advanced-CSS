Web Components

This page has a few of the standard components. These use the shadow DOM to display sub-elements that are not visible in the "light" DOM.

Follow these steps:

  1. Open this page in your browser
  2. Open the web inspector.
  3. Turn on "Show User-Agent Shadow DOM" option.
    • Chrome: In the inspector go to settings, scroll to Elements and check the box: "Show user agent shadow DOM"
    • Safari: Click the `<>` button in the inspector. (the current version of safari seems to have removed htis option and shows the shadow DOM by default?)
  4. Inpsect each of the elements below. Specifically you're looking at the shadow DOM for that element and what you find there.
